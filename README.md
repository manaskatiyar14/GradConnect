# GradConnect 

📌 About Project

GradConnect  is a modern web interface designed to solve the problem of scattered alumni data and weak student–alumni interaction. This project focuses on building a clean, responsive, and user-friendly UI that enables students and alumni to connect, explore opportunities, and engage with their institution digitally.

The main idea is to solve problems like:

* Alumni data getting lost
* No proper communication between students and alumni
* Difficulty for students to find teammates for hackathons or projects


**Since NST does not have anything such we look forward for working on it as a contribution to NST**


# 💡 ONE OF THE UNIQUE FEATURE --

This project also includes a **Team Builder feature**.

Students can:

* Create a team
* Join a team
* Find teammates based on skills

This helps students who struggle to find the right people for hackathons, projects or coding competitions.

---

# 🔥 Features (Frontend Only)

* Simple and clean UI
* Alumni listing page
* User profile UI
* Team Builder page (create/join team)
* Events / opportunities page
* Basic navigation between pages
